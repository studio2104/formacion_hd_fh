Recomendaciones para incorporar proyectos de HD en la materia de Programación para Humanidades Digitales.

Programación para humanidades digitales

// herramientas recomendadas

GitHub Copilot — asistencia contextual en editor y terminal
Asistencia contextual en editor, terminal y GitHub. Sugiere código línea a línea mientras se escribe, reduciendo la fricción entre una pregunta cultural y un prototipo funcional.
GitHub Copilot (GitHub)
Cursor — agente que entiende toda la base de código
Entiende toda la base de código y puede planificar, escribir y revisar archivos completos. Ideal para proyectos como visores de corpus, scrapers o herramientas de archivo donde la lógica abarca varios ficheros.
Cursor (Anysphere)
Replit Agent — de instrucciones a app funcional
Convierte instrucciones en apps o sitios que luego se refinan con pruebas y optimización. Permite pasar rápidamente de una idea humanística a un prototipo desplegado, sin configurar entorno local.
Replit Agent (Replit)
Google Colab — entorno Python en la nube sin instalación
Entorno de notebooks Python que corre directamente en el navegador, sin necesidad de instalar nada. Permite trabajar con datos, visualizaciones y APIs desde cualquier computadora, ideal para un aula con equipos heterogéneos.
Google Colab (Google)

Recomendaciones para incorporar proyectos de HD en la materia Del homo sapiens al humano postdigital.

Del homo sapiens al humano postdigital

// herramientas recomendadas

NotebookLM — análisis de fuentes y resúmenes de audio
Analiza las fuentes que se le suben y puede convertirlas en resúmenes de audio y fichas. Útil para cartografiar marcos teóricos sobre posthumanismo, tecnopolítica o subjetividad digital.
NotebookLM (Google)
Claude — razonamiento y descomposición de problemas complejos
Orientado al razonamiento, análisis y descomposición de problemas complejos. Especialmente útil para trabajar con textos densos y argumentaciones filosóficas o teóricas.
Claude (Anthropic)
ChatGPT Deep Research — investigación documentada
Puede investigar la web, trabajar con archivos cargados y devolver un informe documentado. Para una materia sobre posthumanismo, tecnopolítica o subjetividad digital, sirve para cartografiar marcos teóricos más que para "resolver" la discusión.
ChatGPT Deep Research (OpenAI)
