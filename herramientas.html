Herramientas para crear, editar y gestionar contenido digital en proyectos de humanidades.

// creación y manejo de contenido

ElevenLabs
Generador de voz con IA de alta calidad. Permite crear narraciones, audiolibros y voces sintéticas a partir de texto, con control de tono, emoción e idioma.
audioIA generativa
Figma
Plataforma de diseño de interfaces y prototipado colaborativo en la nube. Ideal para bocetado, sistemas de diseño y creación de prototipos interactivos sin necesidad de programar.
diseñoprototipado
Adobe Firefly
Generador de imágenes y efectos visuales con IA de Adobe. Permite crear ilustraciones, fondos y variaciones visuales a partir de instrucciones en lenguaje natural.
imagenIA generativa
Gemini
Modelo de lenguaje multimodal de Google. Puede procesar texto, imágenes y documentos, útil para investigación, resumen, generación de contenido y análisis de fuentes.
LLMmultimodal
Teachable Machine
Plataforma de Google para entrenar clasificadores de imágenes, sonidos y poses sin escribir código. Permite experimentar con machine learning de forma intuitiva y accesible.
machine learningno-code
Omeka
Plataforma de publicación web para crear colecciones digitales y exhibiciones en línea. Usada por museos, bibliotecas e investigadores para organizar y compartir patrimonio cultural.
archivocolecciones
Descript
Editor de audio y video que funciona como un procesador de texto: editas el medio editando su transcripción. Permite grabar, transcribir, cortar silencios, eliminar muletillas y publicar podcasts o videos de forma sencilla.
audiovideoedición

Herramientas para analizar datos culturales, visualizar patrones y explorar corpus a escala.

// analítica cultural y visualización de datos

Google Colab
Entorno de notebooks Python en la nube, sin instalación. Permite ejecutar análisis de datos, visualizaciones y modelos de IA directamente en el navegador desde cualquier equipo.
Pythonanálisis de datos
Observable
Plataforma de notebooks interactivos para visualización de datos con JavaScript y D3.js. Ideal para crear gráficos exploratorios y dashboards que se actualizan en tiempo real.
visualizaciónJavaScript
Graph Commons
Plataforma para mapear y analizar redes de información de forma visual y colaborativa. Útil para explorar relaciones entre personas, organizaciones, conceptos o audiencias.
redesvisualización
Voyant Tools
Suite de herramientas para análisis y visualización de textos en línea. Permite explorar frecuencias, tendencias, co-ocurrencias y nubes de palabras en corpus de cualquier tamaño.
análisis de textolectura distante
Teachable Machine
Clasificador visual, de sonido y poses entrenable sin código. En contextos de analítica cultural permite identificar patrones visuales o sonoros en colecciones de medios.
machine learningno-code
Baserow
Base de datos colaborativa de código abierto con interfaz visual tipo hoja de cálculo. Útil para organizar corpus, catalogar fuentes o gestionar datos de proyectos de investigación.
base de datosno-code
Gephi
Software de escritorio para análisis y visualización de redes complejas. Permite explorar grafos de relaciones con métricas de centralidad, comunidades y distribución espacial.
redesanálisis
Palladio
Herramienta de Stanford para visualizar datos históricos complejos. Combina mapas, redes, líneas de tiempo y filtros en una interfaz pensada para humanistas sin experiencia en programación.
humanidades digitalesvisualización
Google Ngram Viewer
Rastrea la frecuencia de palabras o frases en millones de libros digitalizados entre 1500 y 2019. Útil para observar cómo cambia el uso del lenguaje a lo largo del tiempo.
corpusanálisis histórico

Herramientas para explorar y analizar grandes volúmenes de texto sin leerlos de forma lineal.

// lectura distante

NotebookLM
Herramienta de Google que analiza documentos cargados por el usuario y genera resúmenes, fichas, respuestas a preguntas y resúmenes de audio. Ideal para trabajar con bibliografía densa o corpus teóricos.
análisis de fuentesresumen
Voyant Tools
Suite de visualización y análisis textual en línea. Permite identificar frecuencias, tendencias y co-ocurrencias en un corpus sin leerlo íntegramente, revelando patrones que la lectura tradicional no detecta.
análisis de textocorpus

Herramientas para crear mapas interactivos, georreferenciar datos y construir líneas de tiempo multimedia.

// mapas

Leaflet
Biblioteca JavaScript de código abierto para crear mapas interactivos en la web. Ligera y flexible, permite añadir capas, marcadores y datos geoespaciales con pocas líneas de código.
JavaScriptmapas web
Mapbox Studio
Plataforma para diseñar mapas personalizados con estilos visuales propios y datos geoespaciales. Ofrece un editor visual y una API potente para integrar mapas en proyectos web.
mapasdiseño
StoryMap JS
Herramienta del Knight Lab para crear narrativas geográficas interactivas. Combina mapas con texto, imágenes y video para contar historias vinculadas a lugares específicos.
narrativano-code
QGIS
Sistema de información geográfica (SIG) de código abierto para analizar y visualizar datos espaciales. Permite trabajar con capas, proyecciones y datos georreferenciados de forma avanzada.
SIGanálisis espacial

// timelines

Timeline JS
Herramienta del Knight Lab para construir líneas de tiempo interactivas y ricas en multimedia a partir de una hoja de cálculo de Google. Sin necesidad de programar.
línea de tiempono-code
Tiki-Toki
Plataforma web para crear líneas de tiempo en 3D con multimedia integrado. Permite visualizar eventos históricos de forma cronológica con imágenes, video y texto enriquecido.
línea de tiempomultimedia

Herramientas para crear narrativas interactivas, ficción ramificada y experiencias de juego digital.

// narrativa de texto interactivo

Twine
Herramienta de código abierto para crear historias interactivas no lineales con bifurcaciones. Ideal para narrativas donde el lector toma decisiones que determinan el rumbo del relato.
texto interactivono-code
Squiffy
Herramienta ligera para crear ficciones interactivas basadas en texto con un lenguaje de marcado sencillo. Genera una página web autocontenida que puede publicarse en línea fácilmente.
texto interactivoficción
Quest
Plataforma para crear y jugar aventuras de texto interactivas. Permite diseñar mundos, objetos y lógica narrativa con una interfaz gráfica o con scripting básico.
aventura de textojuego

// juego digital

Fiero Game Engine
Motor de videojuegos no-code orientado a creadores sin experiencia técnica. Permite diseñar mecánicas, niveles y narrativas de juego mediante bloques visuales y configuración gráfica.
no-codevideojuego
Tabletop Simulator
Plataforma para diseñar y jugar juegos de mesa en un entorno 3D digital. Permite crear prototipos de juegos de mesa con físicas realistas, cartas, fichas y tableros personalizados.
juego de mesaprototipado
