Las Humanidades Digitales representan la intersección entre los métodos computacionales y las preguntas propias de las humanidades. Su finalidad es analizar fenómenos y artefactos culturales desde esta intersección.

Este espacio está diseñado para que explores su significado y aplicaciones.

Para ver detalles de lo que es formación humanística, haz clic en el botón correspondiente en la parte superior.

// el concepto central

En lugar de simplemente leer un libro para averiguar cuántas veces aparece una palabra, un humanista digital podría usar software para escanear 10,000 libros digitalizados en segundos y graficar con qué frecuencia y en qué contexto se usó esa palabra a lo largo de dos siglos.

// métodos comunes en HD

Análisis de texto (minería de datos): uso de algoritmos para encontrar frecuencias de palabras, temas y el tono emocional en colecciones masivas de texto.
Visualización de datos: convertir datos históricos, sociales o geográficos complejos en tablas, gráficos o modelos interactivos.
Mapeo digital: trazar ubicaciones reales o ficticias —como los escenarios de novelas del siglo XIX o las rutas de migración de poblaciones históricas— en mapas interactivos.
Análisis de redes: mapear relaciones y conexiones, como el círculo social de una figura histórica o las citas entre diferentes filósofos.

// herramientas populares para principiantes

No necesitas ser programador para hacer HD. Muchas plataformas están diseñadas específicamente para principiantes:

Voyant Tools: herramienta gratuita basada en la web donde puedes pegar cualquier texto para visualizar instantáneamente patrones de frecuencia de palabras y temas.
Omeka: plataforma web utilizada por museos e investigadores para crear colecciones digitales y exhibiciones interactivas en línea.
TimelineJS: herramienta de código abierto que te permite construir líneas de tiempo interactivas y ricas en contenido multimedia.

// faq

¿Necesito saber programar?
No. Varias herramientas son visuales. Para módulos avanzados introducimos Python, siempre con guías paso a paso.
¿Cuánto tiempo requiere cada módulo?
Cada módulo está pensado para 2–3 horas de trabajo autónomo, más ejercicios opcionales para profundizar.
¿Puedo contribuir con materiales?
¡Sí! El sitio vive en GitHub y aceptamos pull requests. Consulta la sección de colaboración.
