Las Humanidades Digitales integran métodos computacionales con preguntas propias de las humanidades. Este espacio está diseñado para que los docentes exploren estas herramientas y las incorporen en sus aulas.

// faq

¿Necesito saber programar?
No. Varias herramientas son visuales. Para módulos avanzados introducimos Python, siempre con guías paso a paso.
¿Cuánto tiempo requiere cada módulo?
Cada módulo está pensado para 2–3 horas de trabajo autónomo, más ejercicios opcionales para profundizar.
¿Puedo contribuir con materiales?
¡Sí! El sitio vive en GitHub y aceptamos pull requests. Consulta la sección de colaboración.
